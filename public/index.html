WeGetMusic

E-Commerce Web Site.

An E-Commerce Web Site for selling music tracks to be licensed for specific projects, like YouTube videos. We took over the PHP codebase and rewrote most of it, including converting the whole thing to using Doctrine for database connectivity. The net effect has been a higher-performing application that better meets the clients needs.

Linksert

Kiwi Feed

Your news feeds at a glance.

Indicator Keylock

Open Source System Utility.

An indicator that displays the status of the keyboard lock keys in Linux. Created as a solution for keyboards without the common "caps lock" and other keys, this utility was featured on many Ubuntu blogs.

Check out the project on Launchpad.

HTML5 Music Player

Music Web Widget w/ PayPal Integration.

An HTML5 Web Player with PayPal integration to allow the user to sample then purchase tracks. Programmed using AngularJS with a PHP backend and a MySQL Database.

Amazon Smile Redirect
